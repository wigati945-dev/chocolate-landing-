# Choco-boutique-landing

Ini adalah proyek *landing page* interaktif dengan animasi yang dibuat untuk belajar web development / portofolio

## 🚀 Fitur Utama
* **Desain Responsif:** Tampilan rapi di HP, tablet, maupun komputer.
* **Animasi Halus:** Menggunakan pustaka GSAP untuk efek transisi dan kemunculan elemen.
* **Modern UI:** Menggunakan kombinasi warna dan tata letak yang bersih.

## Teknologi yang Digunakan
* **HTML5:** Untuk menyusun struktur halaman.
* **CSS3:** Untuk mengatur gaya, warna, dan tata letak.
* **JavaScript & GSAP:** Untuk mengontrol logika dan animasi interaktif.

## Cara Menjalankan Proyek di Komputer Anda

1. **Clone repositori ini:**
   ```bash
   git clone [Masukkan Link URL GitHub Anda di Sini]
   ```

2. **Buka folder proyek:**
   Buka aplikasi Visual Studio Code, lalu pilih **File > Open Folder** dan arahkan ke folder proyek ini.

3. **Jalankan halaman:**
   Klik kanan pada file `index.html` dan pilih **Open with Live Server** (pastikan Anda sudah menginstal *extension* Live Server di VS Code).
